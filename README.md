# Basic Python Projects

Welcome to the **Basic Python Projects** repository! This collection features beginner-friendly Python applications and games designed to demonstrate foundational coding concepts like loops, conditional logic, functions, and user input handling.

---

## 📁 Repository Structure

```text
├── Cafe Management System/
│   └── main.py
└── Game Using Python/
    ├── calculataor.py
    ├── madlibs.py
    └── scissorpaper.py
```

## 🚀 Projects Included
```text
1. Cafe Management System
Path: Cafe Management System/main.py

Description: A simple command-line interface (CLI) application designed to manage orders at a cafe.
It displays a food/drink menu, takes customer selections, tracks the total bill, and allows adding multiple items sequentially.

2. Calculator
Path: Game Using Python/calculataor.py

Description: A basic calculator application that prompts users for input values and performs standard
arithmetic operations like addition, subtraction, multiplication, and division.

3. Mad Libs Game
Path: Game Using Python/madlibs.py

Description: A classic word-substitution game. The program prompts the player for various parts of speech
(nouns, verbs, adjectives) and plugs them into a hidden story template to generate funny, unpredictable sentences.

4. Rock, Paper, Scissors
Path: Game Using Python/scissorpaper.py

Description: An interactive terminal game where the player competes against an AI computer in a classic match of Rock, Paper, Scissors.
It evaluates inputs and announces the winner or a tie.
```

## 🛠️ How to Run
```text
Prerequisites
Make sure you have Python installed on your computer. You can check by running this command in your terminal:

Bash
python --version

Execution Steps
Clone or download this repository to your local machine.
Open your terminal or command prompt.
Navigate to the directory containing the project you want to try. For example:

Bash
cd "Game Using Python"

Execute the script using Python:
Bash
python scissorpaper.py
```
